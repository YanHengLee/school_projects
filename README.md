## üß† Projects Included

**2. [Medical Data Visualizer](https://github.com/YanHengLee/python_projects/tree/4e25d84ee9e5ad5799d0cbaf79c6d6dfb05ee3ad/My%20Projects/Medical%20Data%20Visualizer):** Analyze medical data, calculates BMI to create overweight indicator, using **categorical plot** and **heatmap** for data analysis and visualization.

[Time Series Visualizer](https://github.com/YanHengLee/python_projects/tree/4e25d84ee9e5ad5799d0cbaf79c6d6dfb05ee3ad/My%20Projects/Time%20Series%20Visualizer):** Analyzes the page view trends of certain forum websites using percentile filtering, generate **line plot**, **bar plot**, and **box plot** for data visualization.

**5. [Arithmetic Arranger](https://github.com/YanHengLee/python_projects/blob/4e25d84ee9e5ad5799d0cbaf79c6d6dfb05ee3ad/My%20Projects/arithmetic_arranger.py):** Formats arithmetic problems vertically, supports addition and subtraction.

**6. [Budget App](https://github.com/YanHengLee/python_projects/blob/4e25d84ee9e5ad5799d0cbaf79c6d6dfb05ee3ad/My%20Projects/budget_app.py):** Created a budget app that tracks deposits, withdrawals, and transfers using **OOP concepts**.

**7. [Days Counter](https://github.com/YanHengLee/python_projects/blob/4e25d84ee9e5ad5799d0cbaf79c6d6dfb05ee3ad/My%20Projects/days_counter.py):** Calculates the number of days between two dates.

**8. [Statistical Calculator](https://github.com/YanHengLee/python_projects/blob/4e25d84ee9e5ad5799d0cbaf79c6d6dfb05ee3ad/My%20Projects/mean_var_std_calculator.py):** Using numpy to calculate mean, variance, standard deviation, max, min, and sum for a 3x3 matrix created from a list of nine numbers.

**9. [Polygon Area Calculator](https://github.com/YanHengLee/python_projects/blob/4e25d84ee9e5ad5799d0cbaf79c6d6dfb05ee3ad/My%20Projects/polygon_area_calculator.py):** Calculates the geomatric property of a square and rectangle, demonstrates the use of inheritance and method overriding.

**10. [Probability Calculator](https://github.com/YanHengLee/python_projects/blob/4e25d84ee9e5ad5799d0cbaf79c6d6dfb05ee3ad/My%20Projects/prob_calculator.py):** Using Monte Carlo simulation logic, calculate the probability of an expected outcome.

**11. [Time calculator](https://github.com/YanHengLee/python_projects/blob/4e25d84ee9e5ad5799d0cbaf79c6d6dfb05ee3ad/My%20Projects/time_calculator.py):** Accepts a starting datetime and a duration to calculate a new datetime by adding the duration to the statring datetime.
